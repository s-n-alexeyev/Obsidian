##  __`cat`__ - объединение файлов

```bash
cat file1 file2 file3 > result
```

----------
## __`cut`__ - удаление части из строчек (-b побайтно, -c посимвольно, -f по полям)

|file.txt|
|--------|
|Winter  |
|Cold    |
|Sad     |

`cut -c 2,3,4,5,10 file.txt`

|результат|
|---------|
|inte     |
|old      |
|ad       |

## __`split`__ - разбиение файла на части (-l по строчкам, -b по байтам)

|file.txt|
|--------|
|1 red   |
|2 yellow|
|3 green |

`split -l 2 file.txt`

|результат|       |
|---------|-------|
|**xaa**  |**xab**|
|1 red    |3 green|
|2 yellow |       |

## __`tr`__ - перевод/удаление символов (-t замена, -d удаление -s объединение)

`echo Hello | tr -t A-Z a-z `

>hello

`echo Hello | tr -t l L `

>HeLLo

`echo Hello | tr -d l`

>Heo

`echo Hello | tr -s l`

>helo

----------
## __`uniq`__ - поиск одинаковых и уникальных строк

|file.txt|
|--------|
|1 red   |
|1 red   |
|2 yellow|
|3 green |

`uniq file.txt`

|результат|
|---------|
|1 red    |
|2 yellow |
|3 green  |

`uniq -c file.txt`

|результат |
|----------|
|2 1 red   |
|1 2 yellow|
|1 3 green |

`uniq -d file.txt`

|результат|
|---------|
|1 red    |

`uniq -u file.txt`

|результат |
|----------|
|1 2 yellow|
|1 3 green |
